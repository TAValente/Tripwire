# Current Phase

Tripwire is in MVP validation.

The first objective is proving that Tripwire consistently finds meaningful strategic issues that humans would otherwise miss.

Acceptable in this phase:

- a command-line interface
- terminal output
- simple repository and diff loading
- narrow, high-confidence review findings

Avoid in this phase:

- dashboards
- web interfaces
- databases
- CI integrations
- GitHub bots
- broad autonomous refactoring behavior
